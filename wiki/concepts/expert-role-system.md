# Expert Role System

## Definition

The expert role system is a way to make AI short drama decisions by separating professional perspectives. Each expert page defines what that role protects, what it produces, and how it reviews work.

## Why It Helps

AI generation often fails because one prompt tries to solve story, visual design, continuity, performance, editing, and sound at the same time. Separating roles reduces ambiguity.

## Default Roles

- [[script-expert]]: story engine.
- [[storyboard-expert]]: shot design.
- [[director-expert]]: staging and performance.
- [[cinematography-expert]]: visual grammar.
- [[editing-expert]]: rhythm and audience attention.
- [[sound-expert]]: audio meaning.
- [[continuity-expert]]: consistency.
- [[prompt-expert]]: repeatable generation.

## Collaboration Rule

Each role should produce concrete artifacts. Avoid vague advice. Good outputs include beat outlines, scene cards, shot lists, prompt packs, edit notes, continuity flags, and review checklists.
