# AI Short Drama Production Workflow

## Goal

Produce a short drama with AI assistance while preserving story clarity, visual continuity, and editability.

## Workflow

### 1. Define Project Frame

Decide:

- Target audience.
- Platform and aspect ratio.
- Episode duration.
- Genre and tone.
- Visual style.
- Toolchain.

Primary pages: [[ai-short-drama]], [[continuity-bible]]

### 2. Build Story Engine

Create:

- Premise.
- Logline.
- Protagonist want.
- Obstacle.
- Reversal.
- Ending hook or payoff.

Primary expert: [[script-expert]]

### 3. Create Scene Cards

For each scene, define:

- Scene purpose.
- Conflict.
- Emotional turn.
- Location.
- Characters.
- Required visual information.

Template: [[scene-card-template]]

### 4. Break Into Shot Units

Convert each scene into shot units:

- Establishing shot.
- Action shot.
- Reaction shot.
- Insert shot.
- Transition shot.

Primary expert: [[storyboard-expert]]

### 5. Lock Continuity

Create or update:

- Character sheet.
- Location sheet.
- Costume and prop notes.
- Visual style rules.

Primary expert: [[continuity-expert]]

### 6. Generate Prompt Packs

For each shot unit, create prompt packages using:

- Stable identity details.
- Shot-specific action.
- Camera and framing.
- Continuity constraints.
- Negative constraints.

Primary expert: [[prompt-expert]]

### 7. Generate And Select Assets

Generate multiple candidates for important shots. Select by:

- Story clarity.
- Character consistency.
- Performance fit.
- Editability.
- Technical quality.

Primary experts: [[director-expert]], [[continuity-expert]]

### 8. Assemble Rough Cut

Build the timeline early. Use placeholders if needed.

Primary expert: [[editing-expert]]

### 9. Sound And Subtitle Pass

Add:

- Dialogue or voice.
- Music cues.
- Ambience.
- Sound effects.
- Subtitles.

Primary expert: [[sound-expert]]

### 10. QA And Delivery

Review against:

- Story clarity.
- Continuity.
- Rhythm.
- Audio.
- Subtitle readability.
- Export requirements.

Template: [[review-checklist-template]]

## Operating Rule

Do not let generated assets define the story. The story and shot list define what assets are acceptable.
